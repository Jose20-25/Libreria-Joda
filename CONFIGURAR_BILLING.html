💳 Configurar Facturación de Google Cloud

⚠️ Cuenta de Facturación Requerida

Para usar Cloud Run (backend en la nube), necesitas configurar una cuenta de facturación en Google Cloud.

No te preocupes: Google Cloud ofrece $300 USD de crédito gratuito para nuevos usuarios.

💰 Costos Estimados

Servicio Costo Mensual Nivel Gratuito
Firebase Hosting $0 10 GB / mes gratis
Cloud Run $0 - $2 2M requests / mes gratis
Cloud Build $0 120 builds / día gratis
Total (uso normal) $0 - $2 / mes Incluido en nivel gratuito

📋 Pasos para Configurar Facturación

1 Ir a la Consola de Facturación

Abrir Consola de Facturación

2 Crear o Vincular Cuenta de Facturación
  • Click en "Crear cuenta de facturación"
  • Ingresa datos de tarjeta (no se cobrará)
  • Activa $300 USD de crédito gratuito (nuevos usuarios)
3 Vincular al Proyecto
  • Selecciona "libreria-joda" en la lista de proyectos
  • Asocia la cuenta de facturación
  • Confirma
4 Ejecutar Despliegue

Una vez configurada la facturación, ejecuta en PowerShell: 

.\deploy.ps1

✅ Alternativa: Solo Firebase Hosting (Sin Backend)

Si prefieres no configurar facturación ahora, puedes:

  1. Desplegar solo el frontend en Firebase Hosting (100% gratis)
  2. Mantener el backend corriendo localmente
  3. Acceder desde tu red local en tablets/móviles

URL Local desde otros dispositivos:
http://192.168.1.25:5001
