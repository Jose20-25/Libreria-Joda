1️⃣ Reiniciar PowerShell

Después de instalar Google Cloud SDK, reinicia PowerShell

2️⃣ Autenticarse en Google Cloud

Se abrirá tu navegador para iniciar sesión con tu cuenta de Google

3️⃣ Configurar el Proyecto

gcloud config set project libreria-joda

4️⃣ Autenticarse en Firebase

firebase login

5️⃣ Ejecutar Despliegue Automático

.\deploy.ps1

Este script hará todo automáticamente (2-3 minutos)
